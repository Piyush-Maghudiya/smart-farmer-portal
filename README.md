# Smart Farmer Community Portal

Welcome to the **Smart Farmer Community Portal** repository! This project consists of two main parts:
1. **Frontend**: A modern React/Vite-based user interface.
2. **Backend**: A robust Node.js / Express / MongoDB application.

---



## 🚀 How to Run the Project

After running the reorganization script, follow these steps to run the applications:

### 1. Backend Server
1. Open your terminal and navigate to the backend folder:
   ```bash
   cd backend
   ```
2. Install the backend dependencies:
   ```bash
   npm install
   ```
3. Configure your environment variables by checking the `.env` file (copied from `.env.example`). Make sure your MongoDB connection URL and port are set.
4. Start the backend server in development mode:
   ```bash
   npm run dev
   ```

### 2. Frontend App
1. Open a new terminal and navigate to the frontend folder:
   ```bash
   cd frontend
   ```
2. Install the frontend dependencies (if not already installed):
   ```bash
   npm install
   ```
3. Start the frontend application:
   ```bash
   npm run dev
   ```
